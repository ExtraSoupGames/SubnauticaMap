# SubnauticaMap
An interactive map of the game Subnautica, showing biomes, biome information, and caves.

## Added
- Biome locations for surface biomes

## Still to add
- Subsurface biomes
- Biome info
- Cave transitions

## Potential future additions
- Wreck locations
- Blueprint finder tool
- Egg finder tool
- Leviathan locations
- Coordinate positioning
